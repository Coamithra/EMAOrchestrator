# Project Runbook

Some intro text that should be ignored.

## Planning

1. **Gather requirements** Review the ticket and collect all acceptance criteria.
2. **Create branch** Create a feature branch from main.

## Implementation

1. **Write the code** Implement the feature following the design spec.
   Make sure to follow coding conventions.
2. **Add tests** Write unit tests covering the new functionality.
3. **Run linter** Ensure code passes all lint checks.

## Review

1. **Self-review** Read through your own diff before requesting review.
2. **Open PR** Create a pull request with a clear description.
